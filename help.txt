<p>Input data in format: {source_abbreviation&sect;date&sect;note&sect;confidence}<br />For {source_abbreviation} use the four-letter abbreviation according the table in yellow help (for example: AAMN or DCRO).<br />For {date} use periodical publication date in the YYYY-MM-DD format (for example: 1930-11-28).<br />For {note} use periodical number (for example: 279)<br />For {confidence} use the number from 0 to 100 (for example: 75)<br />Use string &sect; as a separator of each segment (even if no value is inserted)</p>
<p>Examples:<br />AAMB&sect;&sect;&sect;85<br />DDVO&sect;1930-11-28&sect;279&sect;100<br /><br />If the field contains repeated values, use the concrete value in the beginning of the record and separate it from other information by double equation {==}.
